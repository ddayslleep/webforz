CSS Animation

animation: 名称 | 时长 | 速度曲线 | 延迟 | 次数 | 方向 | 结束状态

@keyframes move {
  0% { transform: translateX(-100px); }
  100% { transform: translateX(100px); }
}

1️⃣ 动画时长（animation-duration）

1秒（快）

animation: move 1s linear infinite;
1s

3秒（中）

animation: move 3s linear infinite;
3s

5秒（慢）

animation: move 5s linear infinite;
5s

2️⃣ 速度曲线（animation-timing-function）

ease 默认

animation: move 2s ease infinite;
ease

linear 匀速

animation: move 2s linear infinite;
linear

ease-in 慢进

animation: move 2s ease-in infinite;
ease-in

ease-out 慢出

animation: move 2s ease-out infinite;
ease-out

ease-in-out 慢进慢出

animation: move 2s ease-in-out infinite;
in-out

3️⃣ 延迟时间（animation-delay）

无延迟

animation: move 2s 0s linear infinite;
0s

延迟1秒

animation: move 2s 1s linear infinite;
1s

延迟2秒

animation: move 2s 2s linear infinite;
2s

4️⃣ 循环次数（animation-iteration-count）

播放1次

animation: move 2s linear 1;
1次

播放3次

animation: move 2s linear 3;
3次

无限循环

animation: move 2s linear infinite;
无限

5️⃣ 动画方向（animation-direction）

normal 正常

animation: move 2s linear infinite normal;
正常

reverse 反向

animation: move 2s linear infinite reverse;
反向

alternate 往返

animation: move 2s linear infinite alternate;
往返

alternate-reverse 反向往返

animation: move 2s linear infinite alternate-reverse;
反向往返

6️⃣ 结束状态（animation-fill-mode）

none 回到原点

animation: move 2s linear none;
none

forwards 停终点

animation: move 2s linear forwards;
forwards

backwards 停起点

animation: move 2s linear backwards;
backwards

both 双向

animation: move 2s linear both;
both
