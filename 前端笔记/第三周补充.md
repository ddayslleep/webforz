# pointer-events 样式

## 一、概念介绍

`pointer-events` 是 CSS 中用于**控制元素是否响应鼠标/指针事件**的核心属性。

简单通俗理解：它决定一个元素能不能被鼠标“点到、碰到、选中”，相当于给元素设置“鼠标穿透权限”。

## 二、常用属性值

### 1. pointer-events: auto;（默认值）

### 2. pointer-events: none;

- 元素**完全不响应任何鼠标事件**，相当于“透明化”鼠标交互。
- 视觉上元素依然存在，但鼠标会直接“穿透”它，忽略它的存在。

# CSS filter 样式

## 一、概念介绍

`filter` 是 CSS 中用于**对元素进行视觉效果处理**的核心属性，相当于给元素“加滤镜”。

简单通俗理解：和手机拍照的滤镜功能一样，无需修改图片本身，就能通过CSS实现模糊、提亮、变色、阴影等效果，适用于图片、文字、div等所有元素。

核心特点：不改变元素的结构和内容，只修改其视觉呈现效果，支持多个滤镜组合使用。

## 二、常用滤镜属性值

格式：`filter: 滤镜1(参数) 滤镜2(参数);`（多个滤镜空格分隔，顺序会影响最终效果）

### 1. blur(`px`)：模糊效果

- 作用：给元素添加高斯模糊，参数为模糊半径（单位：`px`），值越大，模糊效果越明显。

### 2. brightness(%)：亮度调节

- 作用：调整元素的亮度，参数为百分比（%），默认值为100%（正常亮度）。       

### 3. contrast(%)：对比度调节

- 作用：调整元素的对比度（明暗差异），参数为百分比（%），默认值100%。      

### 4. grayscale(%)：灰度效果

- 作用：将元素变成黑白灰效果，参数为百分比（%），默认0%（彩色），100%（完全灰度）。

### 5. opacity(%)：透明度调节

- 作用：调整元素的透明度，参数为百分比（%），默认100%（完全不透明），0%（完全透明）。
- 和 `opacity` 属性的区别：`filter: opacity()` 只影响元素本身，不影响子元素；`opacity` 属性会继承，子元素也会跟着透明。

### 6. drop-shadow(水平偏移 垂直偏移 模糊半径 颜色)：阴影效果

- 作用：给元素添加阴影，和 `box-shadow` 类似，但更适合给透明图片添加阴影（不会出现多余的矩形阴影）。
- 参数说明：  
- 水平偏移（`px`）：阴影左右位置（正数向右，负数向左）         
- 垂直偏移（`px`）：阴影上下位置（正数向下，负数向上）         
- 模糊半径（`px`）：阴影模糊程度，值越大越柔和         
- 颜色：阴影颜色（可写十六进制、`rgba`等）      
- 示例：`filter: drop-shadow(2px 2px 4px rgba(0,0,0,0.3));`（柔和的灰色阴影）

### 7. 组合滤镜

多个滤镜空格组合，实现更丰富的效果，示例：

`filter: blur(2px) brightness(90%) contrast(110%);`（轻微模糊+略暗+高对比度）

实现一张图片的多种滤镜效果切换，直观看到不同滤镜的作用，同时演示组合滤镜的使用。

```html
<!DOCTYPE html>
<html lang="zh-CN">
```

# backdrop-filter属性

## 一、属性概述

backdrop-filter 是CSS3新增的背景滤镜属性，用于对**元素背后的区域**应用图形效果（如模糊、颜色偏移等），而非元素本身或其内容。其核心价值的是实现“背景处理、内容清晰”的视觉效果，是现代UI设计中毛玻璃、亚克力质感的核心实现方式。

与传统filter属性不同，backdrop-filter仅作用于元素后方的背景，不影响元素自身及内部的文字、图标、图片等内容，这也是其区别于filter的关键优势。

## 二、基本语法

### 2.1 基础语法格式

```css
/* 单滤镜效果 */
backdrop-filter: 滤镜类型(参数);

/* 多滤镜效果（用空格分隔） */
backdrop-filter: 滤镜1(参数) 滤镜2(参数);

/* 取消滤镜效果 */
backdrop-filter: none;
```

### 2.2 常用滤镜类型（重点）

backdrop-filter支持多种滤镜效果，其中最常用、最实用的是blur（高斯模糊），其他滤镜可根据需求搭配使用，常用类型如下：

- **blur(`px`)**：高斯模糊，核心用于实现毛玻璃效果，参数为模糊半径（单位`px`），数值越大，模糊效果越强。
- **brightness(%)**：调整背景亮度，100%为正常亮度，大于100%变亮，小于100%变暗。
- **contrast(%)**：调整背景对比度，100%为正常，大于100%对比度增强，小于100%对比度降低。
- **grayscale(%)**：将背景转为灰度，100%为完全灰度，0%为正常颜色。
- **sepia(%)**：将背景转为棕褐色，100%为完全棕褐色，0%为正常颜色。

其中，`backdrop-filter: blur(12px)` 是前端开发中最常用的写法，对应标准毛玻璃效果。

## 生效必备条件

backdrop-filter属性并非单独使用就能生效，需满足以下3个条件，否则滤镜效果会失效或异常：

1. **元素必须设置半透明背景**：不能使用完全不透明的背景色（如`background: #fff`），否则会遮挡背后的内容，无法看到模糊效果；推荐使用`rgba()`设置半透明背景。
2. **元素下方必须有可模糊的内容**：如果元素背后是空白（无背景、无其他元素），则模糊效果无法体现，需确保元素下方有图片、渐变、其他卡片等内容。
3. **避免父级设置overflow: hidden**：如果父元素设置了`overflow: hidden`，可能会裁剪元素背后的模糊区域，导致`backdrop-filter`效果失效（尤其在3D变换场景中）。

# 基线对齐核心概念与原理

基线（Baseline）是CSS排版的核心垂直对齐基准，即文本中字母“站立”的水平线。核心控制属性：`vertical-align`（内联/内联块/表格）、`align-items`（Flex/Grid）

核心痛点：图片下方空白、图标与文字错位。
