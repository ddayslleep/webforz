默认情况下，转换原点是盒子中心点

取值：

### 1.5 多重转换

指一个标签具有多种形态的转换

技巧：先平移再旋转
`transform:translte() rotate();`

**先旋转后平移为什么不行？**
旋转会改变坐标轴向
多重转换：以第一种转换形态的坐标轴为准——先旋转的话，X轴时刻在发生变化，所以平移的X轴就一直变化

**为什么不能分开写？**

拆开写，CSS具有层叠性，旋转会把平移给覆盖

### 1.6 缩放

`transform: scale(X轴缩放倍数，Y轴缩放倍数);`

技巧：

+ 一般只给scale()设置一个值，表示X轴和Y轴等比例缩放
+ 取值大于1表示放大，取值小于1表示缩小

定义：纵横向上的像素点数，单位是px

PC分辨率：

+ 1920*1080
+ 1366*768……

缩放150%——1920/150%   1080/150%

+ 硬件分配率——物理分配率（出厂设置）
+ 缩放调节的分辨率）——逻辑分配率（软件/驱动设置）

pc端和移动端：参考逻辑分配率写代码

## 视口

+ 手机屏幕尺寸不同，网页宽度均为100%
+ 网页的宽度和逻辑分辨率尺寸相同

视口：显示HTML网页的区域，用来约束HTML尺寸

`<meta name="viewport" content="width=device-width, initial-scale=1.0">`
`width=device-width` ——视口宽度=设备宽度

`initial-scale=1.0`——缩放1倍（不缩放）

**没有视口标签，就会使移动端可能无法显示完全部网页**

## 二倍图

概念：设计稿里面每个元素的尺寸的倍数

作用：防止图片在高分辨率吧屏幕下模糊失真

现阶段设计稿都是以设备宽度375px产出设计稿，二倍图设计稿尺寸：750px

媒体查询能够检测视口的宽度，然后编写差异化的CSS样式

> ```css
> @media(媒体特性) {
>             选择器 {
>                 CSS属性
>             }
>         }

> ```css
>  @media (width:375.33px) {

### 2. less

less是一个CSS预处理器，less文件后缀是.less；它扩充了CSS语言，使CSS具备一定的逻辑性、计算能力

浏览器==不识别==less代码，目前网页需要引入对应的CSS文件

注释：

+ 单行注释
  + 语法：//注释内容
  + 快捷键：ctrl+/
+ 块注释
  + 语法：跟CSS的一样
  + 
  + 快捷键：Shift+Alt+A

#### 2.1 运算

+ 加、减、乘直接书写计算表达式
+ 除法需要添加小括号或 .

> ```less
> .box {
>             width: 100+50px;
>             height: 5*32px;
> 
>             width: (100/4px);
>             height: 100./4px;
>         }
> ```

如果表达式有多个单位，最终CSS里面以第一个单位为准

#### 2.2 嵌套

作用：快速生成后代选择器

> ```less
> .父级选择器 {
>             // 父级样式
>             .子级选择器 {
>                 //子级样式
>             }
>         }
> 
>  .father {
>             color: red;
>             .son {
>                 color: blue;
>             }
>         }
> ```
>
> 

可以多级嵌套

&表示的是当前选择器，代码写到谁的打括号里面就表示谁。

应用：配合hover伪类或nth-child结构伪类使用

#### 2.3 变量

概念：容器，存储数据

作用：存储数据，方便使用和修改

语法：

+ 定义变量：@变量名:数据;
+ 使用变量：CSS属性:@变量名;

![image-20260418183939955](https://team-image-zhuoyin.oss-cn-beijing.aliyuncs.com/2025013816/image-20260418183939955.png)

#### 2.4 导入

作用：导入less公共样式文件,可以少生成几个CSS文件，这样只用link一次

语法：导入：@import "文件路径";

提示：如果是less文件可以省略后缀

> ```less
>  @import './base.less';
>  @import './commen';
> ```

#### 2.5 导出

写法：在less文件的第一行添加//out:存储URL

提示：==文件夹==名称后面添加/

> ```less
> //out: ./index.css
> //out:./css/
> ```

**禁止导出**

写法：在less文件第一行添加 `// out:false`
