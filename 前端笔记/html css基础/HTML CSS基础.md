# HTML

## 1.标签语法

- 标签成对出现，中间包裹内容，分为开始标签和结束标签

- <>里面放英文字母（标签名）

- 结束标签比开始标签多/

  **分类**

  1. 双标签：成对出现的标签，需要包裹内容 
     例如：\<strong>文本\</strong>——将文本加粗

  2. 单标签：只有开始标签，没有结束标签  
     例如： \<br>：换行     \<hr>：水平线

  **生成多个标签的快捷方式**

  通过输入标签名*n的方式，使生成n个同类标签

## 2. HTML基本骨架

HTML基本骨架：网页模板

> ```html
> <!DOCTYPE html>
> <html lang="en">
> <head>
>     <meta charset="UTF-8">
>     <meta name="viewport" content="width=device-width, initial-scale=1.0">
>     <title>Document</title>
> </head>
> <body>
>     
> </body>
> </html>
> ```
>
> 

**解释**

+ html: 整个网页
+ head：网页头部，存放给浏览器看的代码
+ body：网页主题，存放给用户看的代码，例如图片、文字等
+ title：网页标题，上图中“网页名称”就应写为——\<title>网页名称\<title>

## 3. 标签的关系（明确代码的书写位置）

1. 父子关系（嵌套关系）
   如2的基本骨架中，\<head>\</head>与\<title>\<title>之间，便是嵌套关系

**写嵌套关系的快捷形式**

嵌套关系中，可以通过输入标签名1>标签名2，来生成嵌套关系的标签，即为标签名1套标签名2

2. 兄弟关系（并列关系）
   如2的基本骨架中，\<head>\</head>与\<body>\</body>之间，便是并列关系

## 4. 注释

定义：对代码的解释和说明

作用：可以增加对代码的可读性

写法：\<!--…-->
被包裹的内容不会在浏览器中显示

快捷方式：Ctrl+/

## 5. 排版标签

### 5.1 标题标签

作用：一般用于新闻标题，文章标题，网页logo，产品名称等

分类：分为1-6级标题，标签名分别用h1-h6表示，为双标签

显示特点：

+ 文字加粗
+ 字号逐渐减小
+ 独占一行（换行）

+ h1一般仅用一次，用于新闻标题等整个网页最重要的部分；其余的没有使用次数的限制

### 5.2 段落标签

作用：一般用于新闻段落、文章段落、产品描述信息等等

标签名：p（双标签）

显示特点：

+ 独占一行
+ 段落之间存在间隙

+ 一个段落中的文字，会随着网页页面的大小，自动调整行数

## 6. 换行与水平线标签

1. 换行标签：br（用于强制换行，调整行间距）
   ==浏览器中不识别Enter键的换行==

2. 水平线标签：hr

## 7. 文本格式化标签

作用：为文本添加特殊格式，以突出重点

常见的文本格式及标签名

| 效果          | 标签名1 | 标签名2 |
| ------------- | ------- | ------- |
| **加粗**      | strong  | b       |
| *倾斜*        | em      | i       |
| <u>下划线</u> | ins     | u       |
| ~~删除线~~    | del     | s       |

**相同点**

==视觉效果都是一样的==，都用于浏览器的渲染

**不同点：**

+ 标签名1是==语义性标签==，自带强调语义，强调重要性，即浏览器/搜索引擎知道你为什么要这样写
+ 标签名2是纯样式标签

## 8. 图像标签

作用：在网页中插入图片

写法：`<img src="图片的URL" alt="替换文本">`

+ src：属性名，用于指定图像的位置和名称（从而插入不同的图片），是\<img>的必须属性。
+ ”图片的URL“——图像的位置和名称

属性（设置图片的特点）：

| 属性   | 作用       | 说明                               |
| ------ | ---------- | ---------------------------------- |
| alt    | 替换文本   | 图片==无法显示的时候==显示的文字   |
| title  | 提示文本   | 鼠标悬停在图片上面的时候显示的文字 |
| width  | 图片的宽度 | 值为数字，没有单位                 |
| height | 图片的高度 | 值为数字，没有单位                 |

+ 浏览器缩放图片，默认等比例缩放

**总结**

`<img src="./cat.jpg" alt="替换文本" title="提示文本">`

+ 属性名=”属性值“
+ 属性写在尖括号里面，标签名后面，标签名和属性之间用空格隔开，不区分先后顺序

## 9. 路径

定义：查找文件时，从起点到终点经历的路线

分类：

1. 相对路径：从==当前文件位置==出发查找目标文件
2. 绝对路径：从==盘符==出发查找目标文件（Mac电脑从根目录出发）

### 9.1 相对路径

写法（写在src的属性值之中）：

+ / 表示进入某个文件夹里面——文件名/

+ . 表示当前文件所在文件夹——./

+ .. 表示当前文件的上一级文件夹——../

  eg：`<img src="./cat.jpg">`

### 9.2 绝对路径

**图片在自己电脑中**

写法：直接在文件夹的路径处，鼠标左键点击，复制粘贴，然后再输入\文件名即可

+ Windows系统中默认\，也可写成/

实际应用中，此种情况均用相对路径，以防分享后，对方无法打开图片的情况

**图片在其他网页中**

写法：找到所需图片，右键，复制图片地址，在粘贴即可

应用场景：友情链接（将文字做成跳转的链接，写对应的在线网址）

## 10. 超链接标签

作用：单击跳转到其他页面（线上网址或者是本地网址）

标签名：a （双标签）

写法：`<a href=""></a>`

+ 被标签包裹的内容：用户将点击的内容，可以是文字或图片
+ href 属性值是跳转地址，是超链接的必须属性
  + 跳转到线上网址直接写该网址
  + 跳转到本地网址，需要用相对路径查找

---

在超链接中添加属性  `target="_blank"`，用新窗口打开页面

**初期，不知道超链接的跳转地址怎么办？**

herf属性值写#，表示空链接，不会跳转

## 11. 多媒体标签

### 11.1 音频标签

写法：`<audio src=""></audio>`

常见属性：

| 属性                | 作用             | 特殊说明                                                     |
| ------------------- | ---------------- | ------------------------------------------------------------ |
| src**（必须属性）** | 音频URL          | 支持格式：**MP3**、Ogg、Wav                                  |
| controls            | 显示音频控制面板 | 用于确定音频是否成功插入                                     |
| loop                | 循环播放         |                                                              |
| autoplay            | 自动播放         | 为了提升用户体验，浏览器一般会**禁用**自动播放功能，所以这个属性写上去并没有什么效果 |

eg：`<audio src="./media/music.mp3"></audio>`

+ 在HTML 5 里，如果属性名和属性值完全一样，可以简写为一个单词。下面两种形式是一样的
  + controls loop autoplay均是这种

> `<audio src="./media/music.mp3" controls></audio>`
>
> `<audio src="./media/music.mp3" controls="controls"></audio>`

### 11.2 视频标签

写法：`<video src=""></video>`

常见属性：

| 属性                | 作用             | 特殊说明                                         |
| ------------------- | ---------------- | ------------------------------------------------ |
| src**（必须属性）** | 视频URL          | 支持格式：**MP4**、WebM、Ogg                     |
| controls            | 显示视频控制面板 |                                                  |
| loop                | 循环播放         |                                                  |
| muted               | 静音播放         |                                                  |
| autoplay            | 自动播放         | 为提升用户体验，浏览器支持在**静音状态**自动播放 |

+ 想要在浏览器中自动播放，必须muted属性与autoplay属性同时存在

## 12. 三大类表格标签

### 12.1 列表（嵌套关系）

作用：布局内容排列整齐的区域

分类：无序列表（使用频率最高）、有序列表、定义列表

#### **1. 无序列表**

作用：布局排列整齐的不需要规定顺序的区域

标签：ul 嵌套 li，ul 是无序列表，li是列表条目

> ```html
> <ul>
>     <li>列表条目1</li>
> </ul>
> ```

注意事项：ul 标签里面只能包裹li 标签；li 标签里面可以包裹任何内容

#### **2. 有序列表**

作用：布局排列整齐的需要规定顺序的区域

标签：ol 嵌套 li，ol 是有序列表，li 是列表条目

> ```html
> <ol>
>     <li>第一项</li>
> </ol>
> ```

注意事项：ol 标签里面只能包裹li 标签；li 标签里面可以包裹任何内容

#### **3.定义列表**

标签：dl 嵌套 dt 和 dd，dl 是定义列表，dt 是定义列表的标题，dd 是定义列表的描述/详情

> ```html
> <dl>
>     <dt>第一条</dt>
>     <dd>这是对第一条的解释</dd>
> </dl>
> ```

dd 标签内的内容，会被浏览器自动缩进

注意事项：dl 标签里面只能包裹dt 和 dd 标签；dt 和 dd 标签里面可以包裹任何内容

### 12.2 表格（嵌套关系）

作用：展示数据

标签：table 嵌套 tr ；tr 嵌套 td / th

| 标签名 | 说明           |
| ------ | -------------- |
| table  | 表格           |
| tr     | 行             |
| th     | **表头**单元格 |
| td     | **内容**单元格 |

 注意：网页中，表格默认没有边框线，需要使用 border 属性来添加边框线——在table 标签里面添加

**表格的结构标签（双标签）**

作用：将内容划分区域，让表格结构更清晰，语义更清晰
肉眼看不到具体效果，但浏览器会清楚这个表格是什么区域（语义标签）

| 标签名 | 含义     | 特殊说明     |
| ------ | -------- | ------------ |
| thead  | 表格头部 | 表格头部内容 |
| tbody  | 表格主体 | 主要内容区域 |
| tfoot  | 表格底部 | 汇总信息区域 |

**合并单元格**

作用：将多个单元格合并成一个单元格，以合并同类信息

合并步骤：

1. 明确合并目标
2. 确保==最上/最左==的单元格，添加属性（取值是数字，表示需要合并的单元格数量）
   + 跨行合并：保留==最上==单元格，添加属性rowspan（直接写在 td 标签内即可）
   + 跨列合并，保留==最左==单元格，添加属性colspan（直接写在 td 标签内即可）
3. 删除被合并的单元格

不能跨结构合并单元格

### 12.3 表单

作用：收集用户信息

使用场景：

+ 登录页面
+ 注册页面
+ 搜索区域

#### **12.3.1 input 标签**

input 标签的type属性不同，则功能不同
`<input type="">`

| type属性值 | 说明                                       |
| ---------- | ------------------------------------------ |
| text       | 文本框，用于输入单行文本，输入什么显示什么 |
| password   | 密码框，输入的内容都以 点 的形式显示       |
| radio      | 单选框                                     |
| checkbox   | 多选框                                     |
| file       | 上传文件                                   |

可以用类似于  `文本框:<input type = "text">`的方式来提示用户这个表单的用途

**占位文本**

作用：提示信息（类似于登录页面中的灰色文字 提示你需要你在该栏输入的信息）

写法：`<input type="…" placeholder="提示信息">`
文本框和密码框都可以使用；placeholder默认未输入的时候是浅灰色文字，用户输入的时候会变成黑色文字

**单选框**

| 属性名  | 作用     | 特殊说明                               |
| ------- | -------- | -------------------------------------- |
| name    | 控件名称 | 控件分组，同组只能选中一个（单选功能） |
| checked | 默认选中 | 属性名和属性值相同，简写为一个单词     |

name 一样，即为一组

**怎么实现一次上传多个文件？**
在input 里添加 multiple 属性，选择文件的时候直接用鼠标右键拖动就行

#### 12.3.2  下拉菜单

标签：select 嵌套 option ，select是下拉菜单整体，option是下拉菜单的每一项

+ selected属性：默认选中（该属性名与属性值相同）
  + 没有此属性，下拉菜单默认显示第一个option包裹的内容

#### 12.3.3 文本域

作用：多行输入文本的表单控件

标签：textarea（双标签）

+ 默认情况下，只书写该标签，右下角有拖拽功能

#### 12.3.4 label标签——增大点击范围

作用：网页中某个标签的说明文本；用其绑定文字和表单控件的关系，增大表单控件的点击范围，从而提升用户的体验（能够通过点击文字来选择）

写法：

1. + label 标签只包裹内容，==不包裹表单控件==
   + 设置 label 标签的 for 属性值和表单控件的 id 属性值==相同==

> ```html
> <input type="radio" id="man">
> <label for="man">男</label>
> ```

2. + 使用 label 标签包裹文字和表单控件，不需要属性

`<label ><input type="radio">女</label>`

+ 支持lable 标签增大点击范围的表单控件：文本框、密码框、上传文件、单选框、多选框、下拉菜单、文本域等

#### 12.3.5 按钮

标签名：button
`<button type=""></button>`

+ type属性值

  | type属性值 | 说明                                             |
  | ---------- | ------------------------------------------------ |
  | submit     | 提交按钮，点击后可以提交到数据到后台（默认功能） |
  | reset      | 重置按钮，点击后将表单控件恢复默认值             |
  | button     | 普通按钮，默认没有功能，一般配合JavaScript使用   |

从外观来看，只有reset可以演示
**包括这些属性的button标签，都需要包裹在form 标签里，才能实现功能**
`<form action=""></form>`
form——表单区域        action——发送数据的地址

#### 12.3.6 无语义的布局标签

作用：布局网页（划分网页区域，摆放内容）

+ div （盒子）：独占一行
+ span ：不换行

#### 12.3.7 字体实体

作用：在网页中显示预留字符（防止浏览器认为是标签）

| 显示结果 | 描述   | 实体名称 |
| -------- | ------ | -------- |
|          | 空格   | \&nbsp;  |
| <        | 小于号 | \&It;    |
| >        | 大于号 | \&gt;    |

+ 在代码中敲键盘的空格，网页只能识别一个，故用字体实体来实现多个空格

# CSS

## 1. 基础知识

定义：层叠样式表，是一种样式表，用来描述HTML文档的呈现（美化内容）

书写位置：title 标签下方，添加 style 双标签， style 标签里面书写CSS代码

书写规则：选择器{属性名 : 属性值}

> ```css
> <style>
>     p{
>         color:red；
>     }
> </style>
> ```

+ 属性名和属性值成对出现——键值对

引入方式：

+ 内部样式表：学习使用
  + 写在 style 标签里面
+ 外部样式表：开发使用
  + 写在单独的CSS文件中
  + 在HTML使用 link 标签引入
    `<link rel="stylesheet" href="路径+文件名后缀">`
    rel：样式表
+ 行内样式：配合JavaScript使用
  + CSS写在标签的 style 属性值里
    `<div style="color: red;">这是标签</div>`

## 2. 选择器

作用：查找标签，设置样式

基础选择器

+ 标签选择器
+ 类选择器
+ id 选择器
+ 通配符选择器

### 2.1 标签选择器

定义：使用==标签名==（不能随意书写）作为选择器

特点：选择==同名标签==设置==相同的样式==，无法差异化同名标签的样式

### 2.2 类选择器

作用：查找标签，差异化设置标签的显示效果
 一般用于设置CSS样式

步骤：

1. 定义类选择器—>  .类名
2. 使用类选择器 —>标签添加 class =“类名”

> ```html
>     <style>
>         /*定义类选择器*/
>         .red{
>             color: red;
>         }
>     </style>
>     <!--使用类选择器-->
>     <div class="red">这是标签</div>
> ```

+ 一个标签只能用一个class

**注意事项**

+ 类名自定义，不要用纯数字或中文，尽量用英文命名，做到见名知意，多个单词可以用 - 连接
+ 一个类选择器可以供多个标签使用
+ 一个标签可以使用多个类名，类名之间用空格隔开

### 2.3 id选择器

作用：查找标签，差异化设置标签的显示效果

场景：id 选择器一般配合JavaScript 使用，很少用来设置CSS样式

步骤：

1. 定义 id 选择器 —> #id名
2. 使用 id 选择器 —> 标签添加 id ="id名"  

规则：同一个 id 选择器在一个页面只能使用一次

### 2.4 通配符选择器

作用：查找页面所有标签，设置相同样式

+ 开发项目初期清除默认样式时使用

写法：*，不需要调用，浏览器自动查找页面的所有标签，设置相同的样式

> ```css
> *{
>             color: red;
>         }
> ```

## 3. 文字控制属性

### 3.1 字体大小

+ 属性名:font-size
+ 属性值：文字尺寸，PC端网页最常用的单位 px（像素）
  + 必须要有单位，不然属性不生效

谷歌浏览器文字有默认大小 16 px

### 3.2 字体粗细

+ 属性名：font-weight

+ 属性值

  + 数字（开发使用）

    | 正常     | 400     |
    | -------- | ------- |
    | **加粗** | **700** |

    

  + 关键字

    | 正常     | normal   |
    | -------- | -------- |
    | **加粗** | **bold** |

### 3.3 字体样式

作用：清除文字默认的倾斜效果

属性名：font-style

属性值：

+ 正常（不倾斜）：normal
+ 倾斜：italic

### 3.4 行高

作用：设置多行文本的间距

属性名：line-height

属性值：

+ 数字 + px
+ 数字（当前标签 font-size 属性值的倍数)
  默认是16px；

行高组成：上间距+文本宽度+下间距

行高的测量方法：从一行文字的最顶端（最底端）量到下一行文字的最顶端（最底端）

**垂直居中**

技巧：行高属性值等于盒子高度属性值（只能是==单行==文字）

### 3.5 字体族

属性名：font-family

属性值：字体名

补充：font-family属性值可以书写多个字体名，各个字体名用==逗号==隔开，执行顺序是==从左向右==依次查找

+ 属性最后设置一个字体族名，最好使用无衬线字体

### 3.6 font 复合属性

使用场景：设置网页文字公共样式

复合属性：属性的简写方式，一个属性对应多个值的写法，各个属性值用空格隔开

书写顺序：是否倾斜  是否加粗  字号/行高  字体==（必须按顺序书写）==
**字号和字体值必须书写，否则font属性不生效**

> ```css
>  div{
>             font: italic 700 30px/2 楷体;
>         }
> ```

## 4.复合选择器

定义：由两个或多个基础选择器，通过不同的方式组合而成

作用：更准确、更高效地选择目标元素（标签）

### 4.1 后代选择器

作用：选中某元素的后代元素  ==所有后代都会被选中==

写法：父选择器 子选择器{CSS属性}，父子选择器之间用空格隔开

> ```html
> <style>
>          div span{
>            color: red;
>         }
>     </style>
> 
> <div>
>         <span>这是文字</span>
>     </div>
> ```

### 4.2 子代选择器

作用：选中某元素的子代元素（最近的子级）

写法：夫选择器 > 子选择器{CSS属性}，父子选择器之间用 > 隔开

> ```html
> <style>
>          div > span{
>            color: red;
>         }
>     </style>
> ```

### 4.3 并集选择器

作用：选中多组标签设置相同的样式

写法：选择器1，选择器2，…，选择器N{CSS属性}，选择器之间用，隔开

> ```html
> <style>
>          div,
>          p,
>          span{
>             color:red;
>          }
>     </style>
> ```

### 4.4 交集选择器

作用：选中同时满足多个条件的元素

写法：选择器1选择器2{CSS属性}，选择器之间连写，没有任何符号

> ```css
> p.box{
>     color:red;
> }
> ```

若交集选择器中有标签选择器，标签选择器必须书写在==最前面==

## 6. CSS特性

概述：化简代码/定位问题，并解决问题

### 6.1 继承性

定义：子级默认继承父级的文字控制属性
eg: 对 body 标签加以文字控制属性，默认 body 标签内的标签都会被添加该文字控制属性（只有子级标签有自己的样式，才不会继承该属性，但其他不具备的属性，仍然会继承）

### 6.2 层叠性

特点：

+ 相同的属性会覆盖：后面的CSS属性覆盖前面的CSS属性
+ 不同的属性会叠加：不同的CSS属性都生效

### 6.3 优先级

定义：也叫权重，当一个标签使用了多种选择器，基于不同种类的选择器的匹配规则

规则：选择器优先级高的样式生效

公式：通配符选择器 < 标签选择器 < 类选择器 < id选择器 < 行内样式 < !important
==选中标签的范围越大，优先级越低==
!important：拥有提权功能，将优先级/权重提到最高，添加在CSS属性中，将其放在需要使用的属性==分号==之前

**叠加计算规则**

叠加计算：若为复合选择器，则需要权重叠加计算

公式：（每一级之间不存在进位）
（行内样式，id 选择器个数，类选择器个数，标签选择器个数）

规则：

+ 从左向右依次比较选个数，同一级个数多的优先级高，若个数相同，则向后比较
+ !important  权重最高
+ 继承权重最低（就算有!important，只要是继承，就是最低）

## 7. Emmt写法

定义：代码的简写方式，输入缩写 vscode 会自动生成对应的代码

+ HTML

  | 类选择器     | 标签名.类名 |

+ CSS：大多数简写方式为属性单词的首字母

## 8. 背景属性

### 1. 拆分写法

#### 1.1 背景图

作用：实现装饰性的图片效果

属性名：background-image (bgi)

属性值：url (背景图URL)

> ```css
> div{
>     width:400px;
>     height:400px;
>     background-image:url(./images/1.png)
> }
> ```

在浏览器中，图片默认是平铺（复制）的效果，需要通过复制来填充满盒子

#### 1.2 背景图平铺方式

属性名：background-repeat（bgr）

| 属性值    | 效果                         |
| --------- | ---------------------------- |
| no-repeat | 不平铺（图片在盒子的左上角） |
| repeat    | 平铺（默认效果）             |
| repeat-x  | 水平方向平铺                 |
| repeat-y  | 垂直方向平铺                 |

#### 1.3 背景图位置

属性名：background-position (bgp)

属性值：水平方向位置 垂直方向位置

+ 关键字

  | 关键字 | 位置 |
  | ------ | ---- |
  | left   | 左侧 |
  | right  | 右侧 |
  | center | 居中 |
  | top    | 顶部 |
  | bottom | 底部 |

+ 坐标（数字+px，==正负==都可以）——正数向右/下，负数向左/上

可以纯英文，也可以纯数字，还可以英文+数字（水平，垂直都需要调）

注意：

+ 关键字取值方式写法，可以颠倒取值顺序
+ 可以只写一个关键字，另一个方向默认为居中；数字只写一个值表示水平方向，垂直方向居中

#### 1.4 背景图缩放

作用：设置背景图大小

属性名：background-size（bgz）

常用属性值：

+ 关键字
  + cover：等比例缩放背景图片以完全覆盖背景区，可能背景图片显示不全
  + contain：等比例缩放背景图片以完全装入背景区，可能盒子有露白
+ 百分比（根据盒子尺寸计算图片大小）
  若写100%，图片的宽度跟盒子宽度一样，图片的高度按照图片比例等比缩放
+ 数字+单位（例如：px）

#### 1.5 背景图固定

作用：使背景不会跟着元素的内容滚动

属性名：background-attachment （bga）

属性值：fixed

### 2. 复合写法

属性名：background（bg）

属性值：背景色 背景图 背景图平铺方式 背景图位置==/==背景图缩放 背景图固定（==空格隔开，不区分顺序==）

## 9. 显示模式

### 9.1 分类

定义：标签（元素）的显示方式

作用：布局网页的时候，根据标签的显示模式选择合适的标签摆放内容

+ 块级元素（比如 div）
  + 独占一行
  + 宽度默认是父级的100%
  + 添加宽高属性生效
+ 行内元素 （比如span）
  + 一行共存多个
  + 尺寸由内容撑开
  + 加宽高不生效
+ 行内块元素（比如图片）
  + 一行共存多个
  + 默认尺寸由内容撑开
  + 加宽高生效

### 9.2 转换显示模式

属性名：display

属性值：

| 属性值       | 效果   |
| ------------ | ------ |
| block        | 块级   |
| inline-block | 行内块 |
| inline       | 行内   |

转块级无法右对齐，因为块元素独占一行
