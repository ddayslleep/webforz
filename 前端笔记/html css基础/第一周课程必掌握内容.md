## 一、HTML 

### 1. 文档结构与基本骨架
- `<!DOCTYPE html>`
- `<html>`、`<head>`、`<body>`
- `<meta charset="UTF-8">`
- `<title>`

### 2. 常用标签

#### 块级元素
- `<h1>` ~ `<h6>`
- `<p>`
- `<div>`
- `<ul>` / `<ol>` / `<li>`
- `<table>` / `<tr>` / `<td>` / `<th>`

#### 行内 / 行内块元素

- `<span>`
- `<a>`（超链接）
- `<img>`
- `<strong>` /  `<em>` 

#### 单标签

- `hr`
- `br`

### 3. 表单相关

- `<form>`
- `<input>`（text / password / radio / checkbox / file）
- `<select>` + `<option>`
- `<textarea>`
- `<button>`（type: submit / reset / button）
- `<label>`（for / 隐式关联）

### 4. 列表与表格
- 无序列表、有序列表
- 表格合并：`rowspan` / `colspan`

### 5. 路径与资源引用

- 相对路径：`./` 、`../`
- 绝对路径：根路径 `/`、盘符路径
- `<img src="">`、`<link href="">`

### 6.字符实体

- `&lt`  
- `&nbsp`

## 二、CSS 基础

### 1. 三种引入方式

- 行内样式（不推荐）
- 内部样式 `<style>`
- 外部样式 `<link>`

### 2. 选择器

#### 基础选择器
- 标签选择器 `div`
- 类选择器 `.class`
- ID选择器 `#id`
- 通配符 `*`

#### 组合选择器
- 后代 `div p`
- 子元素 `div > p`
- 并集 `h1, h2`

#### 伪类选择器

- `:hover`
- `:nth-child()`
- `:first-child` / `:last-child`

#### 伪元素选择器

- `::before`：配合content使用
- `::after`：配合content使用

### 3. 盒模型与布局基础

- `width` / `height`
- 块级 / 行内 / 行内块转换（`display`）

### 4. 背景与图片
- `background-color`
- `background-image`
- `background-size`（cover / contain）
- `background-repeat`
- `background-position`

### 5. 文字与字体样式
- `color`
- `font-size`
- `font-weight`
- `text-align`
- `text-decoration`
- `line-height`
- `text-indent`

### 6. 边框与基础视觉
- `border`
- `border-radius`
