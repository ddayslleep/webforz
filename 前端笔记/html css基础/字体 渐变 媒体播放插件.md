## 如何引入和使用一个新字体

### 1. 最常用、最简单的方式：使用在线字体（Google Fonts / 阿里普惠体）

以Google Fonts为例：

1. 打开字体网站，选一个字体
2. 复制它给你的<link>标签
3. 粘贴到HTML的<head>里
4. 在CSS里用font-family使用

示例：

```html
<head>
  <!-- 引入字体 -->
  <link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Roboto">
  <style>
    p {
      /* 使用字体 */
      font-family: "Roboto", sans-serif;
    }
  </style>
</head>

<head>
  <link href="https://fonts.font.im/css?family=Kirang+Haerang|Oswald" rel="stylesheet">

  <style>
    .title {
      font-family: "Kirang Haerang", cursive;
      font-size: 40px;
      margin: 50px;
    }

    .text {
      font-family: "Oswald", sans-serif;
      font-size: 30px;
      margin: 50px;
    }
  </style>
</head>

<body>
    <!--    sans-serif → 无衬线（干净、现代，最常用）
            serif → 有衬线（像报纸字体）
            cursive → 手写体 -->
  <h1 class="title">Hello 我是 Kirang Haerang 字体</h1>
  <p class="text">我是 Oswald 字体</p>
</body>
```

### 2. 本地字体文件引入（自己有 .ttf / .woff 文件）

用CSS的@font-face：

```css
@font-face {
  font-family: myFont;
  src: url("myFont.ttf");
}

p {
  font-family: myFont;
}

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>本地字体</title>
    <style>
        @font-face {
            font-family: myFont;     
            src: url("z字体.ttf");  
        }

        .my-text {
            font-family: myFont;   
            /* font-size: 30px; */
            color: rgb(8, 0, 0);
        }
    </style>
</head>
<body>
    <h1>默认字体（对比用）</h1>
    <h1 class="my-text">我是自定义本地字体 myFont</h1>
</body>
</html>
```

## 总结

- 引入：<link> 在线引入 或 @font-face 本地引入

- 使用：CSS 里写 font-family

- 注意第二种方法

  字体文件名必须一样：`xxx.ttf`

  字体文件必须和 **.html 文件放在同一个文件夹**

  支持的字体格式：`.ttf` `.woff` `.woff2` `.otf`

## 媒体类的标签（img / audio / video）常用属性 + 事件

### 一、通用媒体标签属性

- src：资源路径
- width / height：宽高
- alt：图片加载失败时的替代文字（仅 img）
- controls：显示播放控件（audio / video）
- autoplay：自动播放
- loop：循环播放
- muted：静音
- poster：视频封面图（仅 video）
- preload：预加载方式（auto / metadata / none）

### 二、媒体示例

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>video 标签</title>
    <style>
        video {
            border: 2px solid #333;
            border-radius: 8px;
        }
    </style>
</head>
<body>
    <h2>最基础的视频（只有src）</h2>
    <video src="./1.mp4" width="600">
    </video>

    <hr>

    <h2>加上播放控件 controls</h2>
    <video src="./1.mp4" width="600" controls>
    </video>

    <hr>

    <h2>自动播放 autoplay muted</h2>
    <video src="./1.mp4" width="600" controls autoplay muted>
    </video>

    <hr>

    <h2>循环播放 loop</h2>
    <video src="./1.mp4" width="600" controls autoplay muted loop>
    </video>

    <hr>

    <h2>视频封面 poster</h2>
    <video src="./1.mp4" 
        width="600" 
        controls 
        poster="./cover.png"
    >
    </video>

    <hr>

    <h2>事件演示</h2>
    <video 
        src="./1.mp4" 
        width="600" 
        controls
        onplay="alert('开始播放啦！')"
        onpause="alert('暂停啦！')"
        onended="alert('播放结束！')"
        onerror="alert('视频加载失败！')"
    >
    </video>

    <div style="margin-top:30px; color:#666;">
        1. src 是视频路径 <br>
        2. controls 显示播放按钮 <br>
        3. autoplay 必须和 muted 一起用 (要求自动播放必须静音)<br>
        4. 事件：播放、暂停、结束、出错 都会触发
    </div>
</body>
</html>
```

#### 1. 播放控制类

- play：开始播放时触发
- pause：暂停时触发
- ended：播放结束时触发

#### 2. 加载类

- loadstart：开始加载资源
- loadeddata：媒体数据加载完成
- canplay：可以开始播放
- error：加载出错

#### 3. 进度类

- timeupdate：播放时间变化时触发

- waiting：缓冲等待时触发

  ```css
  background: linear-gradient(方向, 颜色1 位置, 颜色2 位置, ...);
  ```

| 参数     | 含义               | 可写的值                                         |
| -------- | ------------------ | ------------------------------------------------ |
| **方向** | 渐变从哪里走向哪里 | to top /to right /to bottom right / 角度 (45deg) |
| **颜色** | 渐变的颜色         | 英文、十六进制、rgb 都可以                       |
| **位置** | 颜色从哪里开始渐变 | 0% 20% 50% 100% 等                               |

| 方向写法        | 含义                        |
| --------------- | --------------------------- |
| 不写            | 默认 **从上到下**           |
| to top          | 从下 → 上                   |
| to right        | 从左 → 右                   |
| to bottom right | 左上 → 右下                 |
| 30deg           | 30 度角                     |
| 90deg           | 从左到右（等同于 to right） |

| 颜色写法          | 含义                         |
| ----------------- | ---------------------------- |
| red 0%            | 最开始就是红色               |
| yellow 50%        | 正中间变成黄色               |
| blue 100%         | 最后是蓝色                   |
| red 20%, blue 20% | 20% 位置**硬切色**（无渐变） |

```css
background: radial-gradient(形状 大小 位置, 颜色1, 颜色2);
```

| 参数 | 含义       | 可选值                         |
| ---- | ---------- | ------------------------------ |
| 形状 | 渐变形状   | circle 圆形 /ellipse 椭圆      |
| 大小 | 渐变范围   | closest-side / farthest-corner |
| 位置 | 中心点位置 | center / top / left / 50% 50%  |
| 颜色 | 渐变颜色   | 同线性渐变                     |

```html
<!DOCTYPE html>
<html lang="zh-CN">

<head>
    <meta charset="UTF-8">
    <title>渐变演示</title>
    <style>
        .box {
            width: 300px;
            height: 100px;
            margin: 10px;
            color: white;
            font-size: 18px;
            text-align: center;
            line-height: 100px;
        }

        .box1 {
            background: linear-gradient(red, blue);
        }

        .box2 {
            background: linear-gradient(to right, red, blue);
        }

        .box3 {
            background: linear-gradient(45deg, red, yellow, green);
        }

        .box4 {
            background: linear-gradient(to right, red 0%, yellow 50%, blue 100%);
        }

        .box5 {
            background: radial-gradient(circle, red, yellow, green);
        }
    </style>
</head>

<body>
    <div class="box box1">从上到下</div>
    <div class="box box2">从左到右</div>
    <div class="box box3">45度三色渐变</div>
    <div class="box box4">带位置百分比</div>
    <div class="box box5">径向圆形渐变</div>
</body>

</html>
```
